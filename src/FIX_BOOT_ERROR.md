# প্রজেক্ট রান করার সমস্যা সমাধান (Vite Version Fix)

আপনার বর্তমান এনভায়রনমেন্টে `Vite` এর একটি নতুন ভার্সন (v7.x) ইনস্টল হয়ে আছে যা আপনার `Node.js` (v20.12.1) এর সাথে সাপোর্ট করছে না। আমরা `package.json` ফাইলে এটি ঠিক করেছি, কিন্তু পুরনো `node_modules` ফাইলগুলো এখনো রয়ে গেছে।

এটি সমাধান করতে নিচের ধাপগুলো অনুসরণ করুন:

### ধাপ ১: টার্মিনাল ওপেন করুন
CodeSandbox বা VS Code-এর টার্মিনালে যান।

### ধাপ ২: পুরনো ফাইল ক্লিন করুন এবং নতুন করে ইনস্টল করুন
নিচের কমান্ডটি কপি করে টার্মিনালে পেস্ট করুন এবং এন্টার দিন:

```bash
rm -rf node_modules package-lock.json pnpm-lock.yaml
npm install
```

### ধাপ ৩: সার্ভার চালু করুন
ইনস্টলেশন শেষ হলে আবার সার্ভার চালু করুন:

```bash
npm run dev
```

---
**নোট:** আমি `package.json` ফাইলে `dev` স্ক্রিপ্ট আপডেট করে দিয়েছি যাতে প্রতিবার রান করার আগে এটি অটোমেটিক ডিপেন্ডেন্সি চেক করে। তাই আপনি শুধু `npm run dev` দিলেও কাজ হতে পারে, তবে `rm -rf node_modules` করা সবচেয়ে নিরাপদ।
