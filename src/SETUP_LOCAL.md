# লোকাল সার্ভারে প্রজেক্ট সেটআপ গাইড

এই প্রজেক্টটি আপনার লোকাল কম্পিউটার বা ভিজ্যুয়াল স্টুডিও (VS Code) এনভায়রনমেন্টে চালানোর জন্য নিচের ধাপগুলো অনুসরণ করুন।

## পূর্বশর্ত (Prerequisites)

আপনার কম্পিউটারে অবশ্যই **Node.js** ইনস্টল থাকতে হবে।
- চেক করতে টার্মিনালে লিখুন: `node -v`
- যদি ইনস্টল না থাকে, [nodejs.org](https://nodejs.org/) থেকে LTS ভার্সন ডাউনলোড করে ইনস্টল করুন।

## ইনস্টলেশন ধাপসমূহ

১. **প্রজেক্ট ডাউনলোড/ক্লোন করুন**
   - ফাইলগুলো আনজিপ করুন বা গিট থেকে ক্লোন করুন।
   - টার্মিনাল বা কমান্ড প্রম্পট ওপেন করে প্রজেক্ট ফোল্ডারে যান।

২. **ডিপেন্ডেন্সি ইনস্টল করুন**
   টার্মিনালে নিচের কমান্ডটি চালান:
   ```bash
   npm install
   ```
   এটি `package.json` ফাইলে থাকা সকল লাইব্রেরি ইন্টারনেট থেকে ডাউনলোড করে `node_modules` ফোল্ডারে সেভ করবে।

৩. **এনভায়রনমেন্ট ভেরিয়েবল সেটআপ (ঐচ্ছিক)**
   - `.env.example` ফাইলটির নাম পরিবর্তন করে বা কপি করে `.env` নাম দিন।
   - যদি আপনার Google Maps API Key থাকে, তবে সেটি `VITE_GOOGLE_MAPS_API_KEY=` এর পরে বসান। না থাকলে খালি রাখতে পারেন (ম্যাপ ফিচার কাজ করবে না, কিন্তু অ্যাপ চলবে)।

৪. **ডেভেলপমেন্ট সার্ভার চালু করুন**
   টার্মিনালে লিখুন:
   ```bash
   npm run dev
   ```
   সফল হলে আপনি একটি লিংক দেখতে পাবেন (যেমন: `http://localhost:5173`)। ব্রাউজারে এই লিংকে গেলে অ্যাপটি দেখতে পাবেন।

## বিল্ড তৈরি করা (প্রোডাকশন)

লাইভ সার্ভারে আপলোড করার জন্য বিল্ড ফাইল তৈরি করতে:
```bash
npm run build
```
এটি `dist` ফোল্ডারে অপটিমাইজড ফাইল তৈরি করবে যা আপনি যেকোনো হোস্টিং (Vercel, Netlify, বা cPanel) এ আপলোড করতে পারবেন।

## সাধারণ সমস্যা ও সমাধান

- **'vite' is not recognized**: নিশ্চিত করুন `npm install` সফলভাবে শেষ হয়েছে।
- **Port occupied**: যদি 5173 পোর্ট ব্যস্ত থাকে, Vite স্বয়ংক্রিয়ভাবে অন্য পোর্টে চালু হবে। টার্মিনালে লিংকটি খেয়াল করুন।
- **Missing dependencies**: `rm -rf node_modules package-lock.json` এবং তারপর আবার `npm install` কমান্ড দিন।

কোনো সাহায্যের প্রয়োজন হলে আমাদের টিমের সাথে যোগাযোগ করুন।
