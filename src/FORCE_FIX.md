# গুরত্বপূর্ণ: টার্মিনাল কমান্ড

যেহেতু আপনার সিস্টেমে `pnpm` এর কারণে `vite` এর ভার্সনটি লক হয়ে আছে, তাই `node_modules` ফোল্ডারটি ডিলিট করা আবশ্যক।

**অনুগ্রহ করে নিচের কমান্ডটি কপি করে আপনার টার্মিনালে চালান:**

```bash
rm -rf node_modules package-lock.json pnpm-lock.yaml
```

এরপর:

```bash
npm install
```

এবং শেষে:

```bash
npm run dev
```

যদি আপনি `rm` কমান্ড চালাতে না পারেন, তবে `dev` স্ক্রিপ্টটি আমি আপডেট করে দিয়েছি যাতে এটি অটোমেটিক `vite` রি-ইনস্টল করার চেষ্টা করে। আপনি শুধু `npm run dev` রান করে দেখতে পারেন।
